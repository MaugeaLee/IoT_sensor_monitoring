## 전하 (electric charge) 
- 전기적인 성질을 가지고 있는 물질, 단위 [C]


## 전류 (electric current)
- 단위 시간당 흐른 전하량, 단위 [A]
- I = Q/t(C/s=)[A] ... (1-1)

## 전압 (electric voltage)
- 단위 전하당 한 일, 단위 [V]
- V = W/Q(j/C=)[V] ... (1-3)

## 전력 (power) 
- 단위 시간당 한 일, 단위 [W]
- P = W/t(J/s=)[W] 
- p = W/t = QV/t = VI[W]
